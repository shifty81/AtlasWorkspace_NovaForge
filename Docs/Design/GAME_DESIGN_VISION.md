# NovaForge — Game Design Vision

## Overview

Nova Forge is a **PvE space simulation** for solo play or small groups (2–20 players) with AI wingmates. Built with C++20/OpenGL and the custom NovaForge Engine — featuring procedural ships, FPS interiors, fleet AI, and a living universe.

## Tri-Modal Gameplay

### FPS Mode
- Walk inside ships (procedurally generated interiors)
- EVA (extra-vehicular activity)
- Survival: oxygen, hunger, fatigue
- Fabricators, medical bays, rig loadouts
- Board enemy ships, salvage wrecks

### Flight Mode
- Pilot ships from frigate to titan
- Module activation, target locking, damage types
- Approach, orbit, keep-at-range, warp
- Cinematic warp with meditation audio

### Fleet Command Mode
- RTS-style fleet management
- AI captains with personality axes (aggression, optimism, humor)
- Morale, friendships, grudges
- Formation flying, tactical positioning

## Legend System

Player actions echo across the universe:
- NPCs reference your deeds
- Statues appear in stations
- Myths form around what you've done
- Ancient tech waits to be reverse-engineered
- Your legend influences future world generation

## Visual Style

- Low-poly procedural aesthetic
- Faction-specific ship silhouettes and paint jobs
- Voxel-authoritative terrain and structures
- Custom UI framework (sci-fi cockpit aesthetic)

## Economy

- Background NPC simulation: mining, hauling, trading
- Player-driven market orders
- Manufacturing chains: ore → refined materials → components → ships
- Faction-controlled trade routes and territories
