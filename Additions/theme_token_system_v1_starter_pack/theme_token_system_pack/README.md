# Theme + Token System v1 Starter Pack

This pack provides a starter AtlasUI theme/token implementation for the repo.

## Files
- ThemeTypes.h
- ThemeTokens.h
- Theme.h
- ThemeDefaults.h
- ThemeManager.h
- ThemeDefaults.cpp
- ThemeManager.cpp

## Notes
- Atlas Dark is the default startup theme.
- All development-facing UI should resolve styling through this system.
- No panel or widget should hardcode colors, spacing, radii, or timing once wired in.
