# Equipment GUI Spec v1

## Purpose
Represent the player's R.I.G., worn modules, attachment graph, and dependent unlocks.

## Main Regions
- center silhouette / rig diagram
- slot cluster panels
- equipped item detail panel
- stat summary panel
- dependency / compatibility panel
- quick actions bar

## Required Slot Groups
- core backpack frame
- helmet
- chest left / right
- arm left / right
- leg utility group
- back upper / back lower
- internal systems group
- auxiliary drone/control ports
- mech bridge slot group at late game

## Required Visible Data
- slot occupancy
- lock/unlock state
- power draw
- armor delta
- shield delta
- life support delta
- mobility delta
- linked unlocks
- unmet dependencies
- damage condition

## Key Interactions
- inspect item
- equip / unequip
- compare item
- pin recipe / requirement
- route to crafting or service screen
- filter by compatible modules
- show dependency chain
