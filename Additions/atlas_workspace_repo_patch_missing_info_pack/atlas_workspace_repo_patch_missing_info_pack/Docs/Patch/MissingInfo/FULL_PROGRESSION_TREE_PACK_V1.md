# Full Progression Tree Pack v1

## Structure
Progression is layered, not linear.

### Layer A — Survival Foundation
- life support stability
- inventory access
- hand tools
- shelter/basic fabrication

### Layer B — R.I.G. Expansion
- utility slot unlock
- scanner suite
- shield starter
- helmet overlay
- wrist terminal
- propulsion starter

### Layer C — Base and Automation
- storage indexing
- centrifuge upgrades
- assembly lines
- remote logistics
- drone routing

### Layer D — Vehicle and Ship Operations
- vehicle interface port
- rover/hoverbike operation
- hangar services
- ship systems control
- orbital logistics

### Layer E — Colony and Fleet
- NPC/visitor integration
- skill-specialized crew
- fleet command interface
- advanced production chains

### Layer F — Mech and Nanotech Consolidation
- mech core unlock
- module consolidation
- nanolayer repair/healing
- endgame hybrid systems

## Rule
Each tier should include progression points and prerequisite gates between major unlocks. No major transition should be a single-step jump.
