# Hangar Management Screen Spec v1

## Scope
Screen available when docked or located in a hangar-enabled structure.

## Main Panels
- active craft roster
- selected craft summary
- hull/module condition
- fuel/power/refill state
- cargo summary
- launch readiness checklist
- service actions

## Actions
- assign bay
- refuel/recharge
- repair
- rearm
- move cargo
- set active craft
- route to ship/vehicle service detail screen
